# MNIST AutoEncoder for Deep Learning
#### 以 AutoEncoder 作為預訓練後訓練 MNIST 手寫辨識資料

#### 一、資料集
  1. 來源：採用 MNIST所提供的手寫辨識資料集
  2. 資料格式
      輸入值：28 \* 28 的像素資料
      輸出值：0 ～ 9 的數字輸出

  3. 數量
        訓練集：60000筆
        測試集：10000筆
